About This Demo

This is an Anime Cinema demonstration built with Bootstrap 5 and custom CSS.

All movies, comments, ratings, and form actions are static content for project purposes.

  • Bootstrap components: navbar, cards, badges, forms, list groups, buttons.
  • Bootstrap layout: container, rows, and responsive columns.
  • Custom CSS: theme colors, card style, and details page behavior.
